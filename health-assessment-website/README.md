# Health Assessment Website for Suicide Prevention

This project is a static website designed to provide a health assessment focused on suicide prevention for teens. The website includes an assessment form, resources, and information to support mental health awareness.

## Project Structure

```
health-assessment-website
├── src
│   ├── index.html          # Main landing page
│   ├── assessment.html     # Health assessment form
│   ├── resources.html      # Additional resources and information
│   ├── css
│   │   └── styles.css      # Styles for the website
│   ├── js
│   │   ├── app.js          # Main functionality
│   │   └── assessment.js    # Assessment form functionality
│   └── assets
│       └── fonts           # Font files for typography
├── package.json            # npm configuration file
└── README.md               # Project documentation
```

## Features

- **Assessment Form**: A dedicated page for teens to assess their mental health with a focus on suicide prevention.
- **Resources**: A page providing links to hotlines, support services, and additional information on mental health.
- **Responsive Design**: The website is designed to be accessible on various devices.

## Getting Started

To set up the project locally, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd health-assessment-website
   ```

3. **Open the `index.html` file** in your web browser to view the website.
